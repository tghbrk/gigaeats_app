## 🎨 Design System Specifications

### Color Palette
- **Primary**: #2E7D32 (Green - GigaEats brand)
- **Secondary**: #FF6F00 (Orange - accent)
- **Success**: #4CAF50 (Green - confirmations)
- **Warning**: #FF9800 (Orange - warnings)
- **Error**: #F44336 (Red - errors)
- **Info**: #2196F3 (Blue - information)

### Typography
- **Headers**: Roboto Bold, 18-24px
- **Subheaders**: Roboto Medium, 16-18px
- **Body**: Roboto Regular, 14-16px
- **Captions**: Roboto Regular, 12-14px

### Spacing System
- **Base unit**: 8dp
- **Small**: 8dp
- **Medium**: 16dp
- **Large**: 24dp
- **XLarge**: 32dp

### Component Specifications

#### Cards
- **Elevation**: 2dp
- **Corner radius**: 12dp
- **Padding**: 16dp
- **Margin**: 8dp between cards

#### Buttons
- **Primary**: Filled, 48dp height
- **Secondary**: Outlined, 48dp height
- **Text**: Text only, 48dp height
- **Corner radius**: 8dp

#### Form Fields
- **Height**: 56dp
- **Corner radius**: 8dp
- **Border**: 1dp solid
- **Focus**: 2dp primary color border

#### Chips
- **Height**: 32dp
- **Corner radius**: 16dp
- **Padding**: 8dp horizontal

### Animation Specifications

#### Tab Transitions
- **Duration**: 300ms
- **Curve**: easeInOut
- **Type**: Slide + fade

#### Card Expansions
- **Duration**: 250ms
- **Curve**: easeOut
- **Type**: Height animation

#### Button States
- **Duration**: 150ms
- **Curve**: easeInOut
- **Type**: Color + elevation

#### Loading States
- **Duration**: 1000ms
- **Curve**: linear
- **Type**: Shimmer effect

## 📱 Responsive Breakpoints

### Mobile (< 600dp)
- Single column layout
- Stacked form fields
- Full-width buttons
- Collapsible sections

### Tablet (600dp - 1200dp)
- Two-column layout where appropriate
- Side-by-side form fields
- Larger touch targets
- Enhanced drag-and-drop

### Desktop (> 1200dp)
- Multi-column layout
- Sidebar navigation
- Hover states
- Keyboard shortcuts

This visual design provides a comprehensive, professional interface that enhances the already solid foundation of the GigaEats menu management system.
